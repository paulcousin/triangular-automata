Abstract

This website is about Triangular Automata (TA), which stands for cellular automata in the triangular grid.

  • On this page, you will find an introduction to TA. Navigate the different sections using the arrows. If math expressions do not display correctly, try Chrome or Firefox.
  • In the gallery, each Elementary Triangular Automata (ETA) can be explored in many ways. Use the control panel and follow your curiosity.
  • Long timescale videos are posted on the YouTube channel.
  • You can read the related paper to delve deeper. Some new sections have been added on this website since its publication and are marked by an asterisk in the table of contents.
  • Take a look at the code section for a demonstration of the Mathematica package.
  • Contact me for any question related to this project.
Some rules are stroboscopic so, if you suffer from epilepsy, avoid animations!
