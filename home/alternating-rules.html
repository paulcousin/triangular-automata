Alternating Rules

One can wonder what would happen if, instead of repeatedly applying the same rule to a grid, we switch between different rules.

The first interesting thing to mention was noticed by Pierre-André Noël. If, instead of using the two steps to find the twin of a rule you apply just one of them, you get two rule numbers. Alternating between these rules will destroboscopify a stroboscopic rule (or stroboscopify one that is not). Let us take a glider of rule 53 as an example.


The original rule 53 glider

First, we take the binary rule number with the leading zeros $53=00110101_2$.

  • Swapping ones and zeros: $11001010_2=202$

  • Reversing the digits: $10101100_2=172$


The destroboscopified glider
